Marcadores de Fuentes

B: Se utitliza para poner en negrita al texto

BIG: se utiliza para convertir a una fuente más grande esto es interpretado por el navegador web

I: Se utiliza para convertir el texto a cursiva

s: Se utiliza para poner una raya en el texto

SMALL: Se utiliza para hacer el texto más pequeño

STRIKE: Es el mismo resultado que s, pone una raya en el texto

TT: Se utiliza para far formato de teletipo como se utiliza en los monitores o impresoras antiguas

U: Se utiliza para subrayar una palabra
